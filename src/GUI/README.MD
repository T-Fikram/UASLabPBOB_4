# GUI Module

Folder ini disiapkan khusus untuk **Graphical User Interface (GUI)** yang akan dibuat menggunakan Java (Swing atau JavaFX).

## Tujuan
Folder ini menjadi tempat semua file GUI disimpan agar terpisah dari:
- `model` (struktur data)
- `service` (logika sistem)
- `main` (program utama)

## Catatan
- Kode GUI **belum dibuat**.
- Anggota tim yang bertugas pada bagian GUI dapat menambahkan file `.java` di folder ini.
- Pastikan melakukan import dari package `model` dan `service` saat membangun tampilan GUI.

## Struktur yang Disarankan
- `LoginGUI.java` (opsional)
- `MenuGUI.java`
- `OrderGUI.java`
- `PaymentGUI.java`
- atau file lain sesuai kebutuhan tim.

Silakan membuat file GUI di sini dan menghubungkannya dengan logika sistemnya.
